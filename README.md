# Silent Path
SilentPath — веб-приложение для визуализации транспортного шума и построения тихих маршрутов в городах Казахстана. Пользователи могут просматривать интерактивные карты, анализировать графики шумового времени, оставлять отзывы и заметки, а также получать рекомендации для снижения шумовой нагрузки.
##  Разделы сайта и функции

### Главная
- Краткое описание проекта
- Кнопка «Узнать больше»

### Карта шума
- Интерактивная карта уровня шума для Алматы, Астаны и Шымкента
- Разные формы и цвета зон шума и безопасности
- Легенда карты обновляется при смене города
- Просмотр заметок на карте (только после входа в аккаунт)

### Тихие маршруты
- Выбор точек A и B с возможностью удаления и изменения
- Построение маршрута с обходом зон опасности
- Сравнение обычного и тихого маршрута с динамической статистикой:
  - Расстояние
  - Время
  - Средний шум
  - Оценка комфорта
  - Влияние на здоровье / преимущества

### График шума
- Интерактивные графики шумового времени по городам: Алматы, Астана, Шымкент
- Данные обновляются при выборе города

### О здоровье
- Информация о влиянии транспортного шума на организм
- Ссылки на официальные источники и исследования

### Рекомендации
- Советы для городских служб по снижению шумовой нагрузки
- Динамическое обновление рекомендаций при изменении зон безопасности и опасности
